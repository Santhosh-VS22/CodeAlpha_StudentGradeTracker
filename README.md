# CodeAlpha_StudentGradeTracker
Java-based Student Grade Tracker developed using OOP concepts and ArrayList. Features include adding students, updating marks, searching records, grade calculation, statistics report, and student management.
## Features
- Add Student
- Search Student
- Update Student Marks
- Delete Student
- Grade Calculation
- Average Score Calculation
- Highest Score Detection
- Lowest Score Detection
- GUI Interface using Java Swing
- File Handling for Data Storage
## Technologies Used
- Java
- Swing
- OOP Concepts
- ArrayList
- File Handling
